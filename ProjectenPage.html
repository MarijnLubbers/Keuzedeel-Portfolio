Project 1

Bolhuis Game

Bolhuis game gemaakt met Unity 2D. Project is gemaakt in 4 dagen. We hebben dit gemaakt voor de familie Bolhuis, omdat hun website aan vernieuwing toe was. De bedoeling van het spel is dat je kerstbomen moet planten en verkopen. Je hebt verschillende soorten upgrades die je in de loop van de game kunt kopen om de bomen sneller te laten groeien en zodat je meer score kunt krijgen. 1 game duurt 5 minuten. Na 5 minuten word gekeken wie de meeste score heeft behaald en de winnaar wint een kortingsbon voor de kerstbomen bij Bolhuis.

Project 2

Bamba Blow Showdown

Bamba Blow Showdown is een groot project die ik heb gemaakt met 3 andere studenten. We hebben dit project gemaakt in 4 weken. Dit is het eerste project wat ik heb gemaakt. Het is een 1v1 fighter game gebaseerd op super smash bros. Je hebt meerdere aanvallen en een dash. Je hebt meerdere levels die je kunt spelen. Het is gemaakt met Unity 2D. Het spel duurt totdat iemand word verslagen.

Project 3

2D Platformer

2D Platformer is een Unity 2D project die ik alleen heb gemaakt. Het is gebaseerd op Super Mario waar de bedoeling is om het einde van het leven te halen. Je hebt verschillende items die je kunt oppakken voor score en enemy's die je willen verslaan. Het level duurt ongeveer 3 minuten. Dit project is in 2 weken gemaakt.
